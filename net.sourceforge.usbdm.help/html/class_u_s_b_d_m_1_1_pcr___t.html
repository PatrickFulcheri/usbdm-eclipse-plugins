Detailed Description

template<uint32_t clockMask, uint32_t pcrAddress, int bitNum, PcrValue defPcrValue>
class USBDM::Pcr_T< clockMask, pcrAddress, bitNum, defPcrValue >

Template representing a Pin Control Register (PCR)

Code examples:

// Create PCR type
using PortC_3 = USBDM::Pcr_T<SIM_SCGC5_PORTC_MASK, PORTC_BasePtr, 3, USBDM::DEFAULT_PCR>;
// Configure PCR
PortC_3::setPCR(PinPull_Up,PinDriveStrength_High,PinDriveMode_PushPull,PinIrq_None,PinFilter_None,PinSlewRate_Fast,PinMux_3);
// Disable Port clock
PortC_3::disableClock();
Template Parameters
clockMaskMask for SIM clock register associated with this PCR
pcrAddressPORT to be manipulated e.g. PORTA (PCR array)
bitNumBit number e.g. 3
defPcrValueDefault value for PCR (including MUX value)
