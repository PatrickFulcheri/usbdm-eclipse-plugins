161 /**
162  * @verbatim
163  * +------------------------------+ Simple DMA mode (MLNO = Minor Loop Mapping Disabled)
164  * | Major Loop = | ==================================================
165  * | CITER x Minor Loop |
166  * | | Each DMA request triggers a minor-loop transfer sequence.
167  * | +--------------------------+ |<-DMA Req. The minor loops are counted in the major-loop.
168  * | | Minor Loop | |
169  * | | Each transfer | | The following are used during a minor loop:
170  * | | SADDR->DADDR | | - SADDR Source address
171  * | | SADDR += SOFF | | - SOFF Adjustment applied to SADDR after each transfer
172  * | | DADDR += DOFF | | - DADDR Destination address
173  * | | Total transfer is NBYTES | | - DOFF Adjustment applied to DADDR after each transfer
174  * | +--------------------------+ | - NBYTES Number of bytes to transfer
175  * | +--------------------------+ |<-DMA Req. - Attributes
176  * | | Minor Loop | | - ATTR_SSIZE, ATTR_DSIZE Source and destination transfer sizes
177  * |..............................| - ATTR_SMOD, ATTR_DMOD Modulo --TODO
178  * | | | |
179  * | +--------------------------+ | The number of reads and writes done will depend on NBYTES, SSIZE and DSIZE
180  * | +--------------------------+ |<-DMA Req. For example: NBYTES=12, SSIZE=16-bits, DSIZE=32-bits => 6 reads, 3 writes
181  * | | Minor Loop | | NBYTES must be an even multiple of SSIZE and DSIZE in bytes.
182  * | | Each transfer | |
183  * | | SADDR->DADDR | | The following are used by the major loop
184  * | | SADDR += SOFF | | - SLAST Adjustment applied to SADDR after major loop
185  * | | DADDR += DOFF | | - DLAST Adjustment applied to DADDR after major loop
186  * | | Total transfer is NBYTES | | - CITER Major loop counter - counts how many minor loops
187  * | +--------------------------+ |
188  * | | SLAST and DLAST may be used to reset the addresses to the initial value or
189  * | At end of Major Loop | link to the next transfer.
190  * | SADDR += SLAST | The total transferred for the entire sequence is CITER x NBYTES.
191  * | DADDR += DLAST |
192  * | | Important options in the CSR:
193  * | Total transfer = | - DMA_CSR_INTMAJOR = Generate interrupt at end of Major-loop
194  * | CITER*NBYTES | - DMA_CSR_DREQ = Clear hardware request at end of Major-loop
195  * +------------------------------+ - DMA_CSR_START = Start transfer. Used for software transfers. Automatically cleared.
196  * @endverbatim
197  *
198  * Structure to define a DMA transfer
199  */
