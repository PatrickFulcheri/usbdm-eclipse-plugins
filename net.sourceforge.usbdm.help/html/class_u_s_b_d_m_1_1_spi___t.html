Sets the CTAR value for a given set of communication delays.

Parameters
[in]cssck=> PCS assertion to SCK Delay Scaler
[in]asc=> SCK to PCS negation delay
[in]dt=> PCS negation to PCS assertion delay between transfers
[in]ctarNum=> Index of CTAR register to modify

Note: Determines values for the smallest delay that is not less than specified delays.

Implements USBDM::Spi.
