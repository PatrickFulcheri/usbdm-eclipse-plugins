Detailed Description

template<int channel>
class USBDM::Adc0DiffChannel< channel >

Template class representing an ADC0 differential channel.

Example

// Instantiate the differential ADC0 channels (for ADC0_DM0, ADC0_DP0)
using Adc = USBDM::Adc0DiffChannel<0>;
// Set ADC resolution
Adc.setMode(AdcResolution_11bit_diff );
// Read ADC value
uint32_t value = Adc0.readAnalogue();
Template Parameters
channelADC channel
Examples:
analogue-diff-example.cpp.

Parameters
[in]adcResolutionResolution for converter e.g. AdcResolution_16bit_se

[in]adcSampleInput sample interval. Allows use of higher input impedance sources

static void USBDM::AdcBase_T< Adc0Info >::enableAsynchronousClock ( bool  enable = true)
inlinestaticinherited

Enable ADC internal asynchronous clock source.

Parameters
[in]enabletrue to enable clock, false to disable
Note
It is not necessary to enable the internal clock to use it as an ADC clock source.
If the internal clock is selected, it will be automatically enabled when an ADC conversion is initiated.
However, enabling it beforehand will reduce the latency of the 1st conversion in a sequence.
