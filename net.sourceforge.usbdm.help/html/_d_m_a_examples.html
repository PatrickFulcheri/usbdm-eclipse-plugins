USBDM
Direct Memory Access Controller (DMAC)

Convenience template for DMAC.

Uses the following classes:

  • USBDM::DmaTcd
  • USBDM::DmaMux_T < DmaMuxInfo >
  • USBDM::Dma_T < DmaInfo >

This template is an interface for the Direct Memory Access Controller hardware.
It provides:

  • Definitions for channel mappings
  • Methods to configure the DMAC and DMAMUX

This is a template class with static methods.
It cannot be instantiated.
